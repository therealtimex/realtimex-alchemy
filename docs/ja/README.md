# RealTimeX Alchemy ユーザーガイド

**言語:** [English](../README.md) | [Tiếng Việt](../vi/README.md) | [Français](../fr/README.md) | [日本語](README.md) | [한국어](../ko/README.md) | [Español](../es/README.md)

**RealTimeX Alchemy** へようこそ。このガイドでは、受動的インテリジェンス・エンジンを使用して、ブラウジング時間を高密度のインサイトに「錬成（トランスミュート）」する方法を説明します。

## 目次

1.  ### [はじめに](getting-started.md)
    インストール、初期設定、データベースへの接続方法。
2.  ### [ディスカバリー（発見）: シグナルの探索](discovery.md)
    ディスカバリー・タブの操作、シグナルのフィルタリング、インテリジェンス・フィードの管理。
3.  ### [錬成（トランスミュート）・エンジン](transmute.md)
    生のシグナルを実行可能なインサイト、レポート、またはその他の形式に変換します。
4.  ### [パーソナル・ナレッジ・チャット](chat.md)
    RAGを搭載したチャット・インターフェースを使用して、自分の履歴と対話します。
5.  ### [インテリジェンスとアクティブ・ラーニング](intelligence.md)
    AIスコアリング・システム、「Boost（ブースト）/Dismiss（拒否）」のフィードバック・ループ、ユーザー・ペルソナについて。
6.  ### [設定とソース](configuration.md)
    ブラウザ・プロファイル、LLMプロバイダー（Ollama、OpenAI、Anthropic）、エンジン設定の管理。
7.  ### [トラブルシューティングとサポート](troubleshooting.md)
    一般的なエラー、システム・ログの読み方、ヘルプの入手方法。

---

> [!IMPORTANT]
> **プライバシー優先: 自分のインテリジェンスを所有する**
> RealTimeX Alchemyは「自分のデータベースを持参する（BYO-DB）」モデルに従っています。すべてのブラウジング履歴、抽出されたシグナル、AIが生成したインサイトは、**あなた自身のSupabaseデータベース**に保存されます。当社があなたのデータを見ることはありません。
