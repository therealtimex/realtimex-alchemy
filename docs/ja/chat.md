# パーソナル・ナレッジ・チャット

RealTimeX Alchemy は、**RAG** (Retrieval-Augmented Generation: 検索拡張生成) を搭載したフル機能のチャット・インターフェースを備えており、自分の閲覧履歴と「対話」することができます。

## 仕組み

**Chat** タブで質問すると、Alchemy は AI の一般的なトレーニングデータだけに頼るのではなく、以下のステップを実行します。

1.  ### セマンティック検索
    あなたの質問はベクトル埋め込みに変換されます。
2.  ### シグナルの取得
    Alchemy は Supabase データベース内を検索し、質問に最も関連のあるシグナル（ソースコンテンツ、要約、エンティティ）を見つけ出します。
3.  ### 文脈に応じた回答
    AI は、質問と履歴からの関連する文脈を組み合わせて、正確でソースを引用した回答を生成します。

## チャットの主な機能

### 1. ソースの引用
AI は、回答を生成するためにどのシグナルを使用したかを明示します。これらの引用をクリックすると、元のソースやシグナル・カードを開くことができます。

### 2. セッション管理
チャットはセッションごとに整理されます。Alchemy は会話の内容に基づいて、セッションに適切なタイトル（例：「AI トレンドの合成」や「新しいフレームワークの調査」）を自動的に生成します。

### 3. 持続的なメモリ
チャット・セッションは Supabase に保存されるため、データベースに接続されている任意のデバイスで、中断したところから会話を再開できます。

## より良いチャット結果を得るためのヒント

-   **具体的に**: 「今日何を読んだ？」ではなく、「今日 Ethereum Layer 2 ソリューションについて何を読んだ？」のように質問してください。
-   **要約を依頼する**: 「今週見つけた競合他社 X に関する主要なニュースを要約して。」
-   **合成の質問**: 「React Server Components について読んだ 3 つの異なる記事を比較し、言及されているメリットとデメリットをリストアップして。」
