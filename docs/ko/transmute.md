# 연금술(트랜스뮤트) 엔진

시그널을 채굴하는 것은 전쟁의 절반에 불과합니다. 이러한 시그널을 실행 가능한 지식으로 **연금술(트랜스뮤트)**하는 것이 RealTimeX Alchemy의 진정한 묘미입니다.

## 연금술이란 무엇인가요?

연금술은 "시그널"(독서 기록에서 얻은 원시 인사이트)을 가져와 특수한 파이프라인을 통해 처리하여 새로운 결과물을 생성하는 과정입니다.

### 일반적인 연금술 파이프라인:
-   **핵심 브리핑 (Executive Briefing)**: 여러 관련 시그널을 하나의 주간 보고서로 결합합니다.
-   **시장 인텔리전스 (Market Intelligence)**: 경쟁사에 대한 시그널을 SWOT 분석으로 변환합니다.
-   **뉴스레터 합성 (Newsletter Synthesis)**: 브라우저 기록에서 "나중에 읽기" 다이제스트를 생성합니다.
-   **기술 연구 (Technical Research)**: 기술 논문이나 문서를 기능 사양서로 통합합니다.

## 연금술 엔진 사용 방법

1.  ### 탐색
    사이드바의 **Transmute** 탭으로 이동합니다. 활성 연금술 파이프라인을 확인할 수 있습니다.
2.  ### 엔진 생성
    Alchemy는 활성 카테고리에 대해 "뉴스레터" 파이프라인을 자동으로 생성합니다. 즉, 시그널을 다이제스트로 구성하기 위해 백그라운드에서 항상 작동하고 있습니다.
3.  ### 파이프라인 실행
    **"Run Engine"**을 클릭하여 수동으로 연금술을 트리거할 수 있습니다. 엔진은 다음을 수행합니다:
    -   라이브러리에서 관련 시그널을 스캔합니다.
    -   자신의 "페르소나"에 따라 필터링하여 노이즈를 제거합니다.
    -   LLM을 사용하여 데이터를 합성합니다.
    -   형식화된 보고서나 인사이트를 출력합니다.

## 활성 학습 재정의

인텔리전스 엔진 설정에서 **"Blocked Tags"**(차단된 태그)를 관리하여 엔진이 필터링할 내용을 사용자 정의할 수 있습니다. 특정 주제에 계속 관심이 없다면 블랙리스트에 추가하여 트랜스뮤트 파이프라인에 들어오지 않도록 할 수 있습니다.

---

> [!IMPORTANT]
> 연금술 엔진은 사용자의 "페르소나"를 존중합니다. AI 에이전트에 대한 시그널을 "Boost"하면 기술 연구 파이프라인이 자동으로 해당 주제를 우선시합니다.
