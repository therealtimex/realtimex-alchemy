# 개인 지식 채팅

RealTimeX Alchemy는 자신의 브라우저 기록과 "대화"할 수 있는 **RAG**(Retrieval-Augmented Generation: 검색 증강 생성) 기반의 풀 채팅 인터페이스를 제공합니다.

## 작동 방식

**Chat** 탭에서 질문을 하면 Alchemy는 AI의 일반적인 학습 데이터에만 의존하지 않습니다. 대신 다음 단계를 따릅니다:

1.  ### 시맨틱 검색
    질문이 벡터 임베딩으로 변환됩니다.
2.  ### 시그널 검색
    Alchemy는 Supabase 데이터베이스를 검색하여 질문과 가장 관련이 있는 시그널(소스 콘텐츠, 요약 및 엔티티)을 찾습니다.
3.  ### 문맥에 맞는 답변
    AI는 질문과 기록의 관련 문맥을 함께 제공받아 정확하고 출처가 명시된 답변을 생성합니다.

## 주요 채팅 기능

### 1. 출처 인용
AI는 답변을 생성하는 데 어떤 시그널을 사용했는지 명확하게 언급합니다. 이러한 인용을 클릭하여 원래 소스나 시그널 카드를 열 수 있습니다.

### 2. 세션 관리
채팅은 세션별로 구성됩니다. Alchemy는 대화 주제에 따라 세션에 적합한 제목(예: "AI 트렌드 합성" 또는 "새로운 프레임워크 연구")을 자동으로 생성합니다.

### 3. 영구 메모리
채팅 세션은 Supabase에 저장되므로 데이터베이스에 연결된 모든 장치에서 대화를 중단한 지점부터 다시 시작할 수 있습니다.

## 더 나은 채팅 결과를 위한 팁

-   **구체적으로 질문하세요**: "오늘 무엇을 읽었지?" 대신 "오늘 Ethereum Layer 2 솔루션에 대해 무엇을 읽었지?"라고 질문해 보세요.
-   **요약 요청**: "이번 주에 찾은 경쟁사 X와 관련된 주요 뉴스를 요약해 줘."
-   **합성 질문**: "React Server Components에 대해 읽은 3가지 다른 기사를 비교하고 언급된 장단점을 나열해 줘."
