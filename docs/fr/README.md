---

> [!IMPORTANT]
> **La protection de la vie privée d'abord : possédez votre intelligence**
> RealTimeX Alchemy suit un modèle "Apportez votre propre base de données" (BYO-DB). Tout votre historique de navigation, les signaux extraits et les informations générées par l'IA sont stockés dans **votre propre base de données Supabase**. Nous ne voyons jamais vos données.
