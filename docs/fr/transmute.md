# Le moteur de transmutation

Miner des signaux n'est que la moitié de la bataille. **Transmuter** ces signaux en connaissances exploitables est l'endroit où RealTimeX Alchemy brille vraiment.

## Qu'est-ce que la transmutation ?

La transmutation est le processus consistant à prendre un "Signal" (une information brute de votre historique) et à le traiter via un pipeline spécialisé pour créer une nouvelle sortie.

### Pipelines de transmutation courants :
-   **Executive Briefing** : Combinez plusieurs signaux liés en un seul rapport hebdomadaire.
-   **Market Intelligence** : Transformez les signaux sur les concurrents en une analyse SWOT.
-   **Newsletter Synthesis** : Créez un condensé "Lire plus tard" de votre historique de navigation.
-   **Technical Research** : Consolidez les articles techniques ou la documentation en une spécification de fonctionnalité.

## Comment utiliser le moteur de transmutation

1.  ### Navigation
    Allez dans l'onglet **Transmute** dans la barre latérale. Vous verrez vos pipelines de transmutation actifs.
2.  ### Création du moteur
    Alchemy crée automatiquement des pipelines "Newsletter" pour vos catégories actives. Cela signifie qu'il travaille toujours en arrière-plan pour organiser les signaux en résumés.
3.  ### Exécuter un pipeline
    Vous pouvez déclencher manuellement une transmutation en cliquant sur **"Run Engine"**. Le moteur va :
    -   Scanner votre bibliothèque pour trouver les signaux pertinents.
    -   Filtrer par votre "Persona" (pour éliminer le bruit).
    -   Utiliser un LLM pour synthétiser les données.
    -   Générer un rapport ou une analyse formatée.

## Surcharge de l'apprentissage actif

Vous pouvez personnaliser ce que le moteur filtre en gérant vos **"Blocked Tags"** (Tags bloqués) dans les paramètres de l'Intelligence Engine. Si vous trouvez systématiquement certains sujets inintéressants, les ajouter à la liste noire les empêchera d'entrer dans un pipeline Transmute.

---

> [!IMPORTANT]
> Le moteur Transmute respecte votre "Persona". Si vous "Boostez" les signaux sur les agents d'IA, votre pipeline de "Technical Research" prioritera automatiquement ces sujets.
