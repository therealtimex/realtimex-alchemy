# Découverte : Explorer votre flux d'intelligence

L'onglet **Découverte** est l'endroit où vous parcourez les informations "transmutées" à partir de votre historique de lecture.

## Mise en page de l'onglet Découverte

### 1. Flux de nouveaux signaux
En haut, vous trouverez un flux horizontal des signaux les plus récemment découverts. Ce flux est repliable pour gagner de la place.

### 2. Grille de catégories
Les signaux sont automatiquement catégorisés. Vous verrez des cartes pour des catégories communes telles que :
-   **Stratégie produit interne** : Décisions techniques, feuilles de route.
-   **Intelligence concurrentielle** : Nouvelles sur les rivaux.
-   **Tendances macro** : Changements à l'échelle de l'industrie.
-   **Percées technologiques** : Nouveaux outils ou articles scientifiques.

### 3. Filtres et recherche
-   **Barre de recherche** : Recherchez dans les titres de signaux, les résumés et les tags.
-   **Filtre de catégorie** : Cliquez sur une catégorie dans la grille pour voir uniquement les signaux de ce groupe.
-   **Filtre de score** : (Trouvé dans Logs Système > Signaux Trouvés) Filtrez par score "Impact élevé" (80%+).

## Gestion des signaux

Chaque carte de signal fournit un aperçu du contenu. Cliquer sur une carte ouvre le **Tiroir de signal** :

-   **Résumé exécutif** : Une analyse concise de la source.
-   **Entités** : Personnes, entreprises et technologies mentionnées.
-   **Boost/Dismiss** : Utilisez ces boutons pour entraîner l'IA. "Boost" indique que c'est de haute valeur ; "Dismiss" masque le contenu similaire à l'avenir.
-   **Favori** : Conservez un signal pour une référence permanente.
-   **Notes** : Ajoutez vos propres réflexions ou contexte à un signal.
-   **Lien externe** : Accédez à l'URL source d'origine.

## Génération automatique de catégories

Alchemy est intelligent. Si vous rencontrez fréquemment un sujet spécifique (ex: "Informatique quantique" ou "React Server Components"), il promouvra automatiquement ce tag en tant que catégorie de premier niveau dans votre grille de Découverte.

---

> [!NOTE]
> Les signaux de haute confiance (Score > 80) sont prioritaires dans votre flux pour vous assurer de voir les nouvelles les plus importantes en premier.
